# Security Policy

## Supported Versions

Currently, the following versions are being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report any security vulnerabilities via a private GitHub message or by opening a designated Security Advisory on the GitHub repository. We will aim to respond within 48 hours and provide patches promptly.
