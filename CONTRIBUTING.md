# Contributing to GeminEye

First off, thank you for considering contributing to GeminEye!

## How to Contribute

### Reporting Bugs
If you find a bug in the source code, you can help us by submitting an issue to our [GitHub Repository](https://github.com/RhythmicDias/GeminEye/issues). 

### Submitting a Pull Request
1. Fork the project.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## Code Style
Please adhere to the Clean Code principles and ensure your code is fully modular, adhering to Manifest V3 standards. Maintain Vanilla JS/CSS without bundling tools.
